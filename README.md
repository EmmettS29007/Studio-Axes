# That Time I Got Reincarnated As a Medieval Knight In the Far Far Future and Had to Grab Eggs and Milk to Save the World 

A sci-fi setting where a medieval knight thrown into the future is forced to do basic tasks or forever be banished in this unforgiving realm. For the sake of time, we will only be including one task in this game, being a “level 1”. This task would be getting eggs & milk from the grocery store.
